Hangman Game (Python)

📌 Project Description

This project is a simple text-based Hangman game created using Python.
The player tries to guess a hidden word one letter at a time.
The player has 6 attempts to guess the correct word before the game ends.

🎯 Objective

To build a beginner-friendly Python game using basic programming concepts such as loops, conditions, lists, and strings.

🛠 Technologies Used

- Python
- Random Module
- Console Input/Output

📚 Concepts Used

- random
- while loop
- if-else statements
- strings
- lists

🎮 How the Game Works

1. The program selects a random word from a predefined list.
2. The player guesses one letter at a time.
3. If the letter is correct, it appears in the word.
4. If the letter is incorrect, the number of attempts decreases.
5. The player wins if the word is guessed before the attempts run out.

▶️ How to Run the Program

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code or any Python IDE.
4. Run the program using the command:

python hangman_game.py
